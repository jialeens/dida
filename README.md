## 1.说明

Dida365 client using electron packaging

这是使用electron包装的滴答客户端，解决滴答清单在linux下没有客户端的问题

## 2.安装方法

### yaourt
```shell
yaourt -S dida
```

或
```shell
yaourt -S dida-git
```
### yay

```shell
yay -S dida
```

或
```shell
yay -S dida-git
```
